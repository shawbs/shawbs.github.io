    初步

    阿里云购买服务器ecs实例

    添加安全组

    1. ICMP规则（ping公网IP）
    2. 22端口开放（也可以修改默认22端口，开放其它端口，这里主要是为了ssh远程连接）

    ps: 建议修改默认端口,以下是修改方法

    • 编辑ssh配置文件

      vim /etc/ssh/sshd_config

    • 添加以下

      Port 22
Prot 你的端口

    • 开放你的端口，运行以下

      iptables -A INPUT -m state --state NEW -m tcp -p tcp --dport 你的端口 -j ACCEPT

    • 重启ssh,iptables（centos7.6重启指令是什么，我也不知道，我是直接在阿里控制台，直接重启的服务器）

    连接服务器

    MAC下使用iterm，编写一个.sh文件，输入以下内容

    #!/usr/bin/expect -f
  set port 2121
  set user root
  set host ***
  set password ***
  set timeout -1

  spawn ssh -p$port $user@$host
  expect "*assword:*"
  send "$password\r"
  interact
  expect eof

    配置以下
    item->Preferences->Profiles添加一个启动shell，

    连接

    • 指令
      ssh -p端口 root@ip
    • 快捷: 使用图片中的快捷键(Shortcut key项)command+control+1

    搭建服务器环境

    安装nginx

    wget http://nginx.org/download/nginx-1.13.7.tar.gz
tar zxvf nginx-1.13.7.tar.gz
cd /usr/nginx/nginx-1.13.7

make && make install
cd /usr/nginx/sbin
./nginx -s reload

    开放80防火墙

    iptables -I INPUT -p tcp --dport 80 -j ACCEPT
/etc/init.d/iptables save
/etc/init.d/iptables restart

    测试安装效果

    curl http://localhost

    安装mysql(5.7.25)

    wget -i -c http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql-community-server

    修改默认密码

    set password for root@localhost = password('123456');

//出现Your password does not satisfy the current policy requirements，执行以下指令
set global validate_password_policy=0;
set global validate_password_length=1;
set global validate_password_mixed_case_count=2;

    创建用户 参考

    CREATE USER 'username'@'localhost' IDENTIFIED BY "password";

    示例

    CREATE USER 'dog'@'localhost' IDENTIFIED BY '123456';
CREATE USER 'pig'@'192.168.1.101_' IDENDIFIED BY '123456';
CREATE USER 'pig'@'%' IDENTIFIED BY '123456';
CREATE USER 'pig'@'%' IDENTIFIED BY '';
CREATE USER 'pig'@'%';

    说明：

    • username是你分创建的用户名
    • localhost是你希望能连接该数据库的主机,(localhost只允许本机，ip被允许的具体IP，%允许所有主机)
    • password是用户密码

    授权

    GRANT privileges ON databasename.tablename TO 'username'@'host'

    示例

    GRANT SELECT, INSERT ON test.user TO 'pig'@'%';
GRANT ALL ON *.* TO 'pig'@'%';
GRANT ALL ON maindataplus.* TO 'pig'@'%';

    说明：

    • privileges：用户的操作权限，如SELECT，INSERT，UPDATE等，如果要授予所的权限则使用ALL
    • databasename：数据库名
    • tablename：表名，如果要授予该用户对所有数据库和表的相应操作权限则可用表示，如.*
